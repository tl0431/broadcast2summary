# broadcast2summary v0.3 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Chinese punctuation restoration (funasr), English bilingual transcription (DeepSeek batch translation), output-stage error isolation (wiki/IM failures no longer poison state), and fetch-one mp3 URL support.

**Architecture:** Four independent features sharing one foundational change (Segment.translation + Episode.language fields). Output isolation is a surgical pipeline.py change. Punctuation and translation are new modules (punctuate.py, translate.py) called from pipeline.py after transcription. fetch-one is a 20-line runner.py implementation replacing the existing NotImplementedError stub.

**Tech Stack:** Python 3.11, `funasr>=1.1.0` (new), existing `openai` SDK for DeepSeek translation, `pytest>=8.0`.

**Reference:** [v0.3 Spec](../specs/2026-05-16-v0.3-design.md)

```python
from __future__ import annotations
from .transcribe import Segment

_punct_model = None


def _load_punct_model():
    global _punct_model
    if _punct_model is None:
        from funasr import AutoModel
        _punct_model = AutoModel(model="ct-punc-c", trust_remote_code=True)
    return _punct_model


def punctuate_segments(segments: list[Segment], language: str) -> list[Segment]:
    """Add punctuation to segments. Only runs for zh; returns input unchanged for en.

    Uses funasr ct-punc-c model (~80MB, Chinese + mixed Chinese-English).
    Gracefully falls back to no-op if funasr is unavailable.
    """
    if language != "zh":
        return segments
    if not segments:
        return segments
    try:
        model = _load_punct_model()
    except Exception:
        return segments  # graceful fallback: no punctuation, no crash

    texts = [s.text for s in segments]
    result = model.generate(input=texts)

    return [
        Segment(start=s.start, end=s.end, text=p.get("text", s.text),
                translation=s.translation)
        for s, p in zip(segments, punctuated)
    ]
```

```python
from __future__ import annotations
import json
from .transcribe import Segment


def translate_segments(segments: list[Segment], deepseek_client) -> list[Segment]:
    """Batch-translate English segments to Chinese via DeepSeek.

    Sends all segment texts in ONE API call as a JSON array.
    Returns new Segment list with .translation field populated.
    """
    if not segments:
        return segments

    texts = [s.text for s in segments]
    prompt = (
        "将以下英文播客转写片段翻译成中文。\n"
        "严格按 JSON 数组返回,顺序与输入一致,每条只有 \"t\" 字段:\n\n"
        f"{json.dumps(texts, ensure_ascii=False)}\n\n"
        "返回格式: [{\"t\": \"译文1\"}, {\"t\": \"译文2\"}, ...]"
    )
    raw = deepseek_client.complete(prompt, temperature=0.1)
    translations = json.loads(raw)
    return [
        Segment(start=s.start, end=s.end, text=s.text,
                translation=t.get("t", ""))
        for s, t in zip(segments, translations)
    ]
```
