# broadcast2summary v0.3 Design Spec

**Status:** Draft for review
**Date:** 2026-05-16
**Owner:** TL
**Builds on:** [v0.2 spec](2026-05-15-transcription-speedup.md)

---

## 1. 目标

v0.3 在 v0.2 基础上加四个独立功能,互不依赖,可并行实现:

1. **中文标点恢复** — funasr 批量补标点,解决 Whisper 中文无标点问题
2. **英文播客 + 双语转写** — 英文 feed 自动翻译,markdown 双语字幕式排版
3. **Output 错误隔离** — wiki/IM 失败不再污染 state,本地 markdown 成功即算 success
4. **fetch-one mp3 URL** — 给一个 mp3 直链,走完整流水线

---

## 2. 不在范围

- Speaker diarization(说话人识别)— 推到 v0.4
- fetch-one 网页解析(小宇宙/Apple Podcasts URL 解析 mp3 直链)— 推到 v0.4
- 摘要双语(TL;DR / 要点 同时中英)— 只做转写双语,摘要固定中文

---

## 3. 关键决策

| 决策 | 选择 | 理由 |
|---|---|---|
| 标点触发条件 | `effective_language == "zh"` | funasr 是中文专用;英文 Whisper 自带标点 |
| 标点 cheap mode | 仍然跑 | funasr 本身很快(~5s/万字),不影响 cheap 目标 |
| 翻译触发条件 | `effective_language == "en"` | Whisper 自动检测优先,FeedConfig.language 兜底 |
| 翻译 API | DeepSeek(批量 JSON) | 成本 <¥0.1/期,质量好,已有 client |
| 双语排版 | 原文+译文交替行(双语字幕式) | 可读性最好,可追溯原文 |
| language 来源 | `transcription.language or ep.language or "zh"` | Whisper 检测最准,FeedConfig 兜底 |
| Output 错误隔离 | wiki/IM 各自独立 try/except,失败只记日志 | IM 是通知性质,不是核心产物 |
| fetch-one 元数据 | URL hash 当 guid,可选 --title | 流程现成,不需要网页解析 |

  download → transcribe → summarize → output(local + wiki + IM)

## 5. 实现要点

### 5.1 中文标点恢复(`punctuate.py`)

```python
from __future__ import annotations
from .transcribe import Segment

_punct_model = None


def _load_punct_model():
    global _punct_model
    if _punct_model is None:
        from funasr import AutoModel
        _punct_model = AutoModel(model="ct-punc-c", trust_remote_code=True)
    return _punct_model


def punctuate_segments(segments: list[Segment], language: str) -> list[Segment]:
    """Add punctuation to segments. Only runs for zh; returns input unchanged for en."""
    if language != "zh":
        return segments
    if not segments:
        return segments
    try:
        model = _load_punct_model()
    except Exception:
        return segments  # graceful fallback: no punctuation, no crash

    texts = [s.text for s in segments]
    result = model.generate(input=texts)
    # funasr returns list of dicts with "text" key
    punctuated = result if isinstance(result, list) else [result]
    return [
        Segment(start=s.start, end=s.end, text=p.get("text", s.text),
                translation=s.translation)
        for s, p in zip(segments, punctuated)
    ]
```

**调用位置:** `FasterWhisperBackend.transcribe()` 末尾,opencc 之后:

```python
from .punctuate import punctuate_segments
info_lang_str = info_lang or self.language_hint or "zh"
segs = punctuate_segments(segs, info_lang_str)
```

**模型:** `ct-punc-c`(funasr 官方中文标点模型,~80MB,支持中英混合)

**首次下载:** ~80MB,缓存到 `~/.cache/modelscope/`(funasr 默认路径,不是 HuggingFace)

### 5.2 英文双语转写(`translate.py`)

```python
from __future__ import annotations
import json
from .transcribe import Segment


def translate_segments(
    segments: list[Segment],
    deepseek_client,
) -> list[Segment]:
    """Batch-translate English segments to Chinese via DeepSeek.

    Sends all segment texts in one API call as a JSON array.
    Returns new Segment list with .translation field populated.
    """
    if not segments:
        return segments

    texts = [s.text for s in segments]
    prompt = (
        "将以下英文播客转写片段翻译成中文。\n"
        "严格按 JSON 数组返回,顺序与输入一致,每条只有 \"t\" 字段:\n\n"
        f"{json.dumps(texts, ensure_ascii=False)}\n\n"
        "返回格式: [{\"t\": \"译文1\"}, {\"t\": \"译文2\"}, ...]"
    )
    raw = deepseek_client.complete(prompt, temperature=0.1)
    translations = json.loads(raw)
    return [
        Segment(start=s.start, end=s.end, text=s.text,
                translation=t.get("t", ""))
        for s, t in zip(segments, translations)
    ]
```

**调用位置:** `pipeline.process_episode()` 中,summarize 之后、output 之前:

```python
effective_language = transcription.language or ep.language or "zh"
if effective_language == "en" and deps.deepseek:
    try:
        segments = translate_segments(transcription.segments, deps.deepseek)
    except Exception:
        logger.exception("translation failed for %s — continuing without translation", ep.guid)
        segments = transcription.segments  # fallback: no translation
else:
    segments = transcription.segments
```

**成本:** 1 万字英文 ≈ 1 次 DeepSeek 调用 ≈ ¥0.05

### 5.4 双语 markdown 排版(`output_local.py` + `output_wiki.py`)

`render_markdown` 的完整转写段改为:

```python
for i, seg in enumerate(segments):
    ts = _fmt_hms(seg.start)
    lines.append(f"[{ts}] {seg.text.strip()}")
    if seg.translation:
        lines.append(f"[译] {seg.translation.strip()}")

```

### 5.6 Output 错误隔离(`pipeline.py`)

将当前单一 try/except 拆为三段:

```python
# ---- local markdown (核心产物,失败则整期 failed) ----
try:
    local_path = write_local_markdown(...)
except Exception as e:
    audio_path.unlink(missing_ok=True)
    return _record_failure(deps, ep, "output_local", e, now, mp3_path=None)

# ---- wiki (独立,失败只记日志) ----
wiki_token, wiki_url = None, None
try:
    if deps.lark and deps.lark_folder_token:
        wiki_result = push_summary_to_wiki(...)
        wiki_token = wiki_result.doc_token
        wiki_url = wiki_result.url
except Exception:
    logger.exception("wiki push failed for %s — continuing", ep.guid)

# ---- IM (独立,失败只记日志) ----
try:
    if deps.lark and deps.im_target:
        push_summary_to_im(..., wiki_doc_url=wiki_url)
except Exception:
    logger.exception("IM push failed for %s — continuing", ep.guid)

# ---- success ----
audio_path.unlink(missing_ok=True)
deps.state.record_episode(EpisodeRecord(..., status="success", ...))
```

**行为变化:**
- local markdown 失败 → 整期 failed(核心产物)
- wiki 失败 → 日志,继续,最终 success
- IM 失败 → 日志,继续,最终 success
- mp3 在 local markdown 成功后删除

### 5.7 fetch-one 实现(`runner.py`)

```python
import hashlib

def cmd_fetch_one(url: str, *, cheap: bool = False, title: str | None = None) -> int:
    cfg = _load()
    state_dir = cfg.paths.state_dir
    state_dir.mkdir(parents=True, exist_ok=True)
    state = State(state_dir / "processed.db")
    state.init_schema()

    guid = hashlib.md5(url.encode()).hexdigest()[:16]
    ep = Episode(
        guid=guid,
        title=title or url.split("/")[-1].split("?")[0],
        pub_date=datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ"),
        audio_url=url,
        duration_seconds=0,
        feed_name="manual",
        language="zh",          # Whisper auto-detect will override
        wiki_node_token=None,   # uses root folder fallback
    )
    deps = _build_deps(cfg, state, state_dir, cfg.paths, cheap=_cheap_from_env(cheap))
    result = process_episode(ep, deps=deps)
    if result.success:
        print(f"done: {result.local_path}")
    else:
        print(f"failed at {result.failed_stage}: {(result.error or '')[:200]}")
    return 0 if result.success else 1
```

`cli.py` 的 `fetch-one` 子命令加 `--title` 参数:

```python
fetch_one.add_argument("--title", default=None, help="episode title (optional)")
```

`main()` 里:

```python
if args.cmd == "fetch-one":
    from .runner import cmd_fetch_one
    return cmd_fetch_one(args.url, cheap=args.cheap, title=args.title)
```

## 8. 风险与回滚

- **funasr 安装失败** → `try ImportError` 容错,跳过标点,不 crash
- **funasr 模型下载慢** → 首次 ~80MB,后续缓存;HF_TOKEN 已配置
- **DeepSeek 翻译 JSON 格式错** → `json.loads` 抛异常 → pipeline 记 `output` 阶段失败(translate 在 output 之前,需要在 pipeline 里加 try/except 包住 translate 调用)
- **Segment.translation 字段加入** → frozen dataclass 末尾加默认 None,向后兼容,现有测试不受影响
- **Episode.language 字段加入** → 同上,末尾加默认 "zh",向后兼容
