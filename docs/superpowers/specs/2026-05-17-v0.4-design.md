# broadcast2summary v0.4 Design Spec

**Status:** Draft for review
**Date:** 2026-05-17
**Owner:** TL
**Builds on:** [v0.3 spec](2026-05-16-v0.3-design.md)

---

## 1. 目标

v0.4 在 v0.3 基础上加四个功能:

1. **whisper.cpp + Metal** — 替换 FasterWhisperBackend,利用 Apple Silicon Metal 加速,单期转写从 5-6 min 压到 2-3 min
2. **Speaker diarization** — silero-VAD + wespeaker + k-means 聚类,每个 segment 标注说话人
3. **Speaker 身份识别** — 摘要时顺带让 DeepSeek 推断 SPEAKER_XX → 真名映射,3 态分类(CONFIRMED / UNCERTAIN / UNKNOWN)
4. **launchd + fetch-one 网页解析** — 替换 cron 支持开机补跑;小宇宙 + Apple Podcasts URL 直接跑完整流水线

---

## 2. 不在范围

- whisper.cpp 以外的 ASR 后端(OpenAI Whisper API 等)
- 超过 6 个说话人的聚类优化
- 说话人跨期记忆(每期独立识别)
- fetch-one 支持 Spotify / 其他平台
- 通用 RSS URL 解析(已有 `feeds add` 命令)

---

## 3. 关键决策

| 决策 | 选择 | 理由 |
|---|---|---|
| Python 绑定 | `pywhispercpp`(PyPI) | 无需手动 cmake,Metal 自动启用 |
| 保留 FasterWhisperBackend | 是,配置切换 | 向后兼容,cheap 模式可选 |
| 默认 backend | `whisper_cpp` | Metal 更快 |
| VAD | silero-VAD(ONNX) | ~1MB,语言无关,无 PyTorch 推理依赖 |
| Speaker embedding | wespeaker ResNet34-zh | ~30MB,中文专训,ONNX |
| 聚类 | scikit-learn k-means | 已有依赖,足够 2-6 人播客 |
| Speaker 身份识别时机 | 摘要时顺带(同一次 DeepSeek 调用) | 零额外 API cost,LLM 看全文更准 |
| Speaker 置信度 | 纯规则 3 态(不让 LLM 自评) | LLM 自评校准差;规则可测试可解释 |
| Transcript 格式 | `[SPEAKER_XX]` 先标 ID,摘要后替换真名 | 解耦 diarization 和身份识别 |
| launchd 触发时间 | 23:00 | 睡前触发,早上看结果 |
| fetch-one URL 类型 | mp3 直链(现有)+ 网页 URL(新增) | 向后兼容 |
| Apple Podcasts 解析 | iTunes lookup API | 无需爬页面,稳定 |

## 5. 实现要点

### 5.1 WhisperCppBackend

```python
class WhisperCppBackend:
    """Apple Metal-accelerated transcription via whisper.cpp."""

    def __init__(self, *, cheap: bool = False, language_hint: str | None = None,
                 n_threads: int = 4, convert_traditional: bool = True):
        self.model_size = "small" if cheap else "large-v3-turbo"
        self.language_hint = language_hint
        self.n_threads = n_threads
        self.convert_traditional = convert_traditional
        self._model = None

    def _load(self):
        if self._model is None:
            from pywhispercpp.model import Model
            self._model = Model(self.model_size, n_threads=self.n_threads)
        return self._model

    def transcribe(self, audio_path: Path) -> TranscriptionResult:
        model = self._load()
        raw_segs = model.transcribe(
            str(audio_path),
            language=self.language_hint or "auto",
        )
        segs = [
            Segment(start=s.t0 / 100.0, end=s.t1 / 100.0, text=s.text.strip())
            for s in raw_segs
        ]
        # opencc + punctuation 走同一后处理路径
        info_lang = self.language_hint or "zh"
        if self.convert_traditional and info_lang == "zh":
            from opencc import OpenCC
            cc = OpenCC("t2s")
            segs = [Segment(start=s.start, end=s.end, text=cc.convert(s.text))
                    for s in segs]
        from .punctuate import punctuate_segments
        segs = punctuate_segments(segs, info_lang)
        return TranscriptionResult(language=info_lang, segments=segs)
```

**`diarization: false` 行为:** 跳过 `diarize_audio` 和 `apply_speaker_names`。所有 segments 的 `speaker_id` 和 `speaker_name` 保持 None。摘要 prompt 不含 `speaker_names` 字段要求。render_markdown 不显示 speaker 标注。

### 5.2 Speaker Diarization (`diarize.py`)

```python
@dataclass(frozen=True)
class SpeakerTurn:
    speaker_id: str   # "SPEAKER_00", "SPEAKER_01", ...
    start: float
    end: float


def diarize_audio(audio_path: Path, *, max_speakers: int = 6) -> list[SpeakerTurn]:
    """Detect speaker turns using silero-VAD + wespeaker embeddings + k-means."""
    import numpy as np
    from sklearn.cluster import KMeans

    # 1. VAD: detect speech segments
    speech_segments = _run_vad(audio_path)

    # 2. Embedding: extract speaker vector per speech segment
    embeddings = _extract_embeddings(audio_path, speech_segments)

    # 3. Cluster: determine number of speakers (2..max_speakers)
    n_speakers = _estimate_n_speakers(embeddings, max_speakers)
    labels = KMeans(n_clusters=n_speakers, n_init=10).fit_predict(embeddings)

    return [
        SpeakerTurn(speaker_id=f"SPEAKER_{labels[i]:02d}",
                    start=seg[0], end=seg[1])
        for i, seg in enumerate(speech_segments)
    ]


def align_speakers(segments: list[Segment],
                   turns: list[SpeakerTurn]) -> list[Segment]:
    """Assign speaker_id to each Segment by maximum time overlap."""
    result = []
    for seg in segments:
        best = _find_best_turn(seg, turns)
        result.append(Segment(
            start=seg.start, end=seg.end, text=seg.text,
            translation=seg.translation,
            speaker_id=best.speaker_id if best else None,
        ))
    return result
```

**silero-VAD 调用方式:** ONNX Runtime,不需要 PyTorch。
**wespeaker 调用方式:** ONNX Runtime,加载 `wespeaker-resnet34-zh` 模型(中英通用)。

**3 态分类规则:**

```python
INTRO_PHRASES = ["我是", "我叫", "I'm ", "I am ", "my name is"]
ADDRESS_WINDOW_SEC = 5.0

def apply_speaker_names(
    segments: list[Segment],
    speaker_names: dict[str, str | None],   # from DeepSeek summary JSON
) -> list[Segment]:
    """Apply CONFIRMED/UNCERTAIN/UNKNOWN labels from summary speaker_names."""
    opening = [s for s in segments if s.start < 180]
    labels = {}
    for sid, name in speaker_names.items():
        if name is None:
            labels[sid] = (None, "UNKNOWN")
        elif _confirmed_self_intro(sid, name, opening):
            labels[sid] = (name, "CONFIRMED")
        elif _confirmed_by_address(sid, name, opening):
            labels[sid] = (name, "CONFIRMED")
        elif name in " ".join(s.text for s in opening):
            labels[sid] = (name, "UNCERTAIN")
        else:
            labels[sid] = (None, "UNKNOWN")

    result = []
    for seg in segments:
        sid = seg.speaker_id
        if sid and sid in labels:
            name, state = labels[sid]
            display = (f"{name}?" if state == "UNCERTAIN"
                       else name if state == "CONFIRMED"
                       else sid)
            result.append(Segment(
                start=seg.start, end=seg.end, text=seg.text,
                translation=seg.translation,
                speaker_id=sid, speaker_name=display,
            ))
        else:
            result.append(seg)
    return result


def _confirmed_self_intro(sid, name, opening):
    segs = [s for s in opening if s.speaker_id == sid]
    for seg in segs:
        if any(p in seg.text for p in INTRO_PHRASES) and name in seg.text:
            return True
    return False


def _confirmed_by_address(sid, name, opening):
    for i, seg in enumerate(opening):
        if seg.speaker_id != sid and name in seg.text:
            window = [s for s in opening[i+1:] 
                      if s.start - seg.start <= ADDRESS_WINDOW_SEC]
            if any(s.speaker_id == sid for s in window):
                return True
    return False
```

### 5.4 URL Resolver (`url_resolver.py`)

```python
@dataclass(frozen=True)
class EpisodeMeta:
    title: str
    audio_url: str
    pub_date: str
    duration_seconds: int


def resolve_url(url: str) -> EpisodeMeta:
    """Resolve a podcast episode webpage URL to mp3 + metadata."""
    if "xiaoyuzhou" in url or "xyzfm" in url:
        return _resolve_xiaoyuzhou(url)
    elif "podcasts.apple.com" in url:
        return _resolve_apple(url)
    else:
        raise ValueError(f"Unsupported URL (supported: xiaoyuzhou, podcasts.apple.com): {url}")


def _resolve_xiaoyuzhou(url: str) -> EpisodeMeta:
    import httpx, re, json
    html = httpx.get(url, follow_redirects=True, timeout=30).text
    m = re.search(r'<script type="application/ld\+json">(.*?)</script>', html, re.DOTALL)
    if not m:
        raise ValueError("No ld+json found in xiaoyuzhou page")
    data = json.loads(m.group(1))
    return EpisodeMeta(
        title=data["name"],
        audio_url=data["associatedMedia"]["contentUrl"],
        pub_date=data.get("datePublished", ""),
        duration_seconds=_parse_iso_duration(data.get("duration", "PT0S")),
    )


def _resolve_apple(url: str) -> EpisodeMeta:
    import httpx, re
    m = re.search(r'id(\d+).*?[?&]i=(\d+)', url)
    if not m:
        raise ValueError(f"Cannot parse podcast/episode ID from URL: {url}")
    podcast_id, episode_id = m.group(1), m.group(2)
    resp = httpx.get(
        f"https://itunes.apple.com/lookup?id={podcast_id}"
        f"&media=podcast&entity=podcastEpisode&limit=50",
        timeout=30,
    ).json()
    for r in resp.get("results", []):
        if r.get("kind") == "podcast-episode" and str(r.get("trackId")) == episode_id:
            return EpisodeMeta(
                title=r["trackName"],
                audio_url=r["episodeUrl"],
                pub_date=r.get("releaseDate", ""),
                duration_seconds=r.get("trackTimeMillis", 0) // 1000,
            )
    raise ValueError(f"Episode {episode_id} not found in Apple podcast {podcast_id}")


def _parse_iso_duration(s: str) -> int:
    """Parse ISO 8601 duration (PT1H23M45S) to seconds."""
    import re
    m = re.match(r'PT(?:(\d+)H)?(?:(\d+)M)?(?:(\d+)S)?', s)
    if not m:
        return 0
    h, mn, sec = (int(x or 0) for x in m.groups())
    return h * 3600 + mn * 60 + sec
```

## 8. 风险与回滚

- **pywhispercpp 安装失败** → `backend: faster_whisper` 回退
- **Metal 不可用** → pywhispercpp 自动回退 CPU
- **silero-VAD/wespeaker ONNX 模型下载失败** → `diarization: false` 跳过
- **Speaker 识别全部 UNKNOWN** → 正常 fallback,transcript 不带 speaker 标注
- **小宇宙 ld+json 结构改变** → url_resolver 抛 ValueError,提示用户用 mp3 直链
- **Apple iTunes API 限流** → httpx timeout → fetch-one 报错,保留 mp3 直链用法
